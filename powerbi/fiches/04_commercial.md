# Board 4 — Commercial & Relation Client

**Objectif** : piloter l'**activité commerciale**, la segmentation et les portefeuilles conseillers.
**Public** : conseillers, directeurs commerciaux.

## Indicateurs
| KPI | Mesure | Définition |
|-----|--------|------------|
| Clients actifs | `Nb Clients Actifs` | `top_client_actif = O` |
| Prospects | `Nb Prospects` | Relation `PR` |
| AUM Total | `AUM Total` | Encours sous gestion |
| AUM Moyen | `AUM Moyen` | AUM / nombre de clients |
| Taux actifs | `Taux Clients Actifs` | Actifs / total |
| Clients | `Nb Clients Totaux` | Base active |

## Visuels
- **Colonnes « Clients par type › groupe »** : segmentation par type de client ; **drill-down** Type → Groupe de risque.
- **Combo « AUM (barres) & taux d'actifs (ligne) par agence »** : performance des agences sur deux axes.
- **Treemap « Clients par groupe de risque »** : poids relatif de chaque groupe.
- **Table « Portefeuille par conseiller »** : nom, type, conseiller, agence, actif, AUM.

## Filtres
Type client · Agence (globaux) · Groupe de risque · Actif (O/N).

## Lecture / insights
- Comparer **AUM moyen** entre agences : potentiel de montée en gamme.
- Le combo révèle les agences à fort AUM mais **faible taux d'actifs** = risque de dormance / attrition.
- La table par conseiller permet la répartition et le suivi de charge des portefeuilles.
